# Proteo-R1 Paper Website

This repository hosts the official project website for the ICML paper **Proteo-R1** using GitHub Pages.

## Quick Start

1. Edit `index.html` and replace placeholders:
   - paper title
   - author names and affiliations
   - paper links (OpenReview/ArXiv/PDF/Code)
   - abstract, method, and results content
2. Commit and push to GitHub.
3. Enable GitHub Pages:
   - Go to **Settings -> Pages**
   - Set **Source** to `Deploy from a branch`
   - Select branch `main` and folder `/ (root)`
4. Your site will be available at:
   - `https://<your-github-username>.github.io/<repo-name>/`

If your repository is named `<your-github-username>.github.io`, then the site URL is:
`https://<your-github-username>.github.io/`.
